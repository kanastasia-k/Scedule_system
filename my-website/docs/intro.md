# Обзор системы расписания

Система **University Scheduler** предназначена для автоматизации процесса составления учебного расписания, управления бронированием аудиторий и оперативного информирования пользователей об изменениях.

## Ключевой функционал
*   **Автоматическая генерация**: Создание сетки занятий на основе учебных планов и доступности преподавателей.
*   **Умное бронирование**: Поиск свободных помещений с учетом необходимого оборудования.
*   **Интеграции**: Поддержка NLP-модуля для обработки пожеланий в свободной форме и уведомления в мессенджерах.

4.  **API Reference**: Техническое описание эндпоинтов и асинхронных интеграций.
