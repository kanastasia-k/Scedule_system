## Описание экранов

### 1. Главное меню и навигация
Навигация в приложении адаптируется под роль авторизованного пользователя:
*   **Для студента/преподавателя:** Главное меню содержит кнопки: Бронирование аудиторий, Расписание звонков, Расписание учебных групп, Расписание преподавателей, Справочная информация, Таблица оценок. При открытии приложения по умолчанию открывается экран расписания (своей группы или своего графика).
*   **Для сотрудника учебного отдела:** В главное меню добавляется приоритетный пункт «Составление расписания», который открывается по умолчанию при входе в систему.

### 2. Составление и изменение расписания (Учебный отдел)
Поток генерации расписания состоит из нескольких шагов:
1.  **Загрузка данных:** Экран с предложением перетащить файл (CSV/Excel) с учебными планами или выбрать его через проводник. Нажатие кнопки «Загрузить».
2.  **Настройка параметров:** Экран выбора дисциплин (например, Математический анализ, ТВиМС).
3.  **Фильтрация:** Для каждой дисциплины открывается окно выбора фильтров (оснащенность компьютерами, наличие лабораторного оборудования). Также задаются общие ограничения (недоступные аудитории, учет индивидуальных планов).
4.  **Генерация:** После нажатия «Сформировать» алгоритм обрабатывает данные и выводит готовое расписание с кнопками «Изменить» и «Сохранить».

### 3. Просмотр расписания и поиск замены
*   **Экран расписания:** Отображает календарную сетку (Пн-Вс). При выборе дня выводится список пар с указанием времени, дисциплины, аудитории (например, Б23-101) и преподавателя.
*   **Замена (Преподаватель/Учебный отдел):** При нажатии на конкретную пару открывается карточка замены. Пользователь может выбрать нового преподавателя или аудиторию из выпадающих списков, после чего нажать «Сохранить».

### 4. Бронирование аудиторий (Студент/Преподаватель)
*   **Выбор фильтров:** Пользователь задает параметры для поиска (оснащенность компьютерами, лабораторное оборудование, нужный временной интервал).
*   **Результаты поиска:** Система отображает экран со списком свободных аудиторий, подходящих под критерии.
*   **Подтверждение:** Нажатие на кнопку «Забронировать аудиторию» закрепляет помещение за пользователем.
