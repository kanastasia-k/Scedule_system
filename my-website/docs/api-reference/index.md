В данном разделе описаны контракты взаимодействия с серверной частью системы автоматизированного составления расписания. 

Спецификация разработана в формате OpenAPI 3.0 и описывает основные методы интеграции:
*   Загрузка учебных планов и настройка генерации.
*   Получение и корректировка черновиков расписания.
*   Поиск свободных ресурсов (аудиторий и преподавателей).

Полная интерактивная версия спецификации доступна в специальном разделе: **[Перейти к OpenAPI документации](/api/)**.
